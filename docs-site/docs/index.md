# Personal Academic Manager

Selamat datang di dokumentasi **Personal Academic Manager** (PAM) - sistem manajemen tugas dan belajar yang dirancang khusus untuk mahasiswa.

## Apa itu PAM?

PAM adalah aplikasi web personal yang membantu Anda:

- **Mengelola Tugas** dengan template, fase, dan health monitoring
- **Melacak Sesi Belajar** dan membangun konsistensi (target 5 sesi/minggu)
- **Capture Ideas** dengan Inbox untuk semua materi dan referensi
- **Upload File ke Cloud** dengan versioning dan backup otomatis
- **Weekly Review** untuk evaluasi progress mingguan
- **Email Reminder** untuk tugas yang mendesak

## Prinsip Penggunaan

PAM dibangun dengan filosofi sederhana:

:::tip Prinsip Utama
**"Akademik bukan soal bakat, tapi soal sistem dan konsistensi."**
:::

Tiga pilar utama PAM:

1. **Focus** - Top 3 task prioritas di dashboard
2. **Execute** - Log sesi belajar & track progress
3. **Finish** - Health monitoring & smart reminders

## Fitur Unggulan

### Health Status & Attention Flags

Setiap tugas memiliki **health score** (0-100) berdasarkan progress dan deadline:

- **Aman** (70-100): On track, hijau
- **Rawan** (40-69): Perlu perhatian, kuning
- **Bahaya** (0-39): Urgent, merah

### Start Early H-7

Tugas yang dimulai minimal **7 hari sebelum deadline** mendapat tracking khusus untuk analisis perencanaan.

### Weekly Consistency

Target **5 sesi belajar per minggu** dengan progress visual dan streak daily.
