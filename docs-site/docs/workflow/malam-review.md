# Malam Review

Rutinitas malam (20:00-22:00) untuk tracking progress.

## Best Time

**20:00-22:00** (setelah sesi belajar malam, sebelum tidur)

## Workflow (10 menit)

### 1. Log Sesi Hari Ini (3 menit)

Jika belum log sesi:
- Klik **"Log Sesi"** di dashboard
- Isi durasi & task terkait
- Submit

### 2. Update Progress (5 menit)

- Buka task yang dikerjakan hari ini
- Toggle **checklist items** yang sudah done
- Progress otomatis update

### 3. Cek Dashboard (2 menit)

- Lihat apakah **Top 3** berubah
- Cek **Study Progress**: X/5 sesi minggu ini
- Lihat **Daily Streak**
