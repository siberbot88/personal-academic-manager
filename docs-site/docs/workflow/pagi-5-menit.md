# Pagi 5 Menit

Rutinitas pagi untuk cek prioritas harian.

## Workflow (5 menit)

### 1. Buka Dashboard (1 menit)

Lihat **Fokus Utama** (Top 3 cards):
- Slot 1: Deadline terdekat
- Slot 2: Belum disentuh
- Slot 3: Risiko tinggi

### 2. Cek Next Action (2 menit)

Setiap card menampilkan **Next Action**:
- Checklist item pertama yang belum done
- Atau fallback: "Tentukan next action"

Prioritaskan action yang bisa diselesaikan kurang dari 30 menit.

### 3. Mental Planning (2 menit)

Tentukan:
- **Task utama** hari ini (dari Top 3)
- **Kapan** akan kerjakan (time blocking)
- **Target** minimal: 1 sesi belajar (50m)

## Contoh

```
08:05 - Buka PAM Dashboard
08:06 - Cek Top 3:
  Slot 1: Makalah Algoritma (Next: Buat outline)
  Slot 2: Laporan Basis Data (Next: Cari 5 jurnal)
  Slot 3: Presentasi Jarkom (Health: Bahaya, merah)
  
08:07 - Pilih: Presentasi Jarkom (prioritas karena bahaya)
08:08 - Plan: 19:00-20:00 (1 sesi deep work)
08:10 - Done!
```

## Tips

- **Jangan** langsung kerjakan tugas di pagi hari
- **Fokus** hanya pada planning & prioritas
- **Komit** pada 1-2 task utama (jangan overload)
