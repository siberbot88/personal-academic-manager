# Promote Material

Cara promote Inbox Item menjadi Material.

## Kapan Promote?

Inbox Item yang **valuable untuk referensi jangka panjang** sebaiknya dipromote jadi Material.

Contoh:
- Link jurnal/artikel penting
- Note summary lecture
- File/dokumen referensi

## Cara Promote

### 1. Dari Inbox List

Navigasi: **Inbox** → klik item → **"Promote to Material"**

### 2. Form Auto-Fill

Form Material otomatis terisi:
- **Judul**: dari konten Inbox
- **Tipe**: auto-detect (Link/Note/File)
- **Tags**: copy dari Inbox tags

Edit jika perlu, lalu **Simpan**.

### 3. Inbox Item Dihapus

Setelah promote, Inbox item otomatis dihapus (sudah jadi Material).

## Example Flow

```
Inbox:
  Konten: https://journal.acm.org/algorithm-paper
  Tags: algoritma, jurnal

↓ Promote

Material:
  Judul: Algorithm Paper - ACM Journal
  Tipe: Link
  URL: https://journal.acm.org/algorithm-paper
  Course: Algoritma & Pemrograman
  Tags: algoritma, jurnal
```
