# Inbox Capture

Quick capture untuk ideas, links, dan notes.

## Apa itu Inbox?

**Inbox** adalah tempat menangkap semua **ide, link, referensi** yang muncul secara spontan, tanpa perlu langsung kategorisasi.

Filosofi: **Capture fast, organize later.**

## Cara Menggunakan

### 1. Quick Capture

Navigasi: **Inbox** → **Buat Inbox Item**

Form minimal:
- **Konten**: teks bebas (link, note, ide)
- **Tags**: kategori (opsional)

Klik **Simpan** → selesai!

### 2. Process Later

Ketika ada waktu (mis: malam review):
- Buka **Inbox**
- Review semua items
- **Promote** yang penting:
  - Jadi **Material** (untuk referensi jangka panjang)
  - Jadi **Task** (action item)
- **Hapus** yang tidak relevan

## Use Cases

### Link Jurnal/Artikel

```
Konten: https://journal.example.com/paper123
Tags: metodologi-penelitian, jurnal
```

Promote → Material (type: Link)

### Ide Tugas

```
Konten: Buat presentasi tentang Dijkstra Algorithm
Tags: jarkom, ide-tugas
```

Promote → Task

### Random Note

```
Konten: Dosen bilang tambahkan section literature review di BAB 2
Tags: makalah, reminder
```

Keep di Inbox atau promote → Task checklist item.
