# Membuat Tugas

Panduan lengkap membuat dan mengelola tugas di PAM.

## Cara Membuat Tugas

### Dari Dashboard

1. Klik **"Buat Task"** di header
2. Isi form dengan minimal:
   - **Judul**: nama tugas yang jelas
   - **Mata Kuliah**: pilih dari dropdown
   - **Deadline**: tanggal terakhir pengerjaan

### Form Lengkap

| Field | Wajib | Keterangan |
|-------|-------|------------|
| Judul | ✅ | Nama tugas, mis: "Makalah Algoritma" |
| Mata Kuliah | ✅ | Pilih course dari dropdown |
| Deadline | ✅ | Tanggal & waktu (HH:MM) |
| Template | ❌ | Auto-split phases (Makalah, Laporan, Presentasi) |
| Ukuran | ❌ | Estimasi effort (S/M/L/XL) |

:::tip Gunakan Template
Template otomatis membuat phases & checklist items berdasarkan best practice untuk tugas seperti Makalah atau Laporan Akhir.
:::

## Template & Auto-Split

### Template Makalah

Otomatis membuat 3 phases:

1. **Riset & Referensi** (3 hari)
   - [ ] Tentukan topik
   - [ ] Cari 5-8 jurnal/buku
   - [ ] Buat outline

2. **Penulisan Draft** (5 hari)
   - [ ] Tulis pendahuluan
   - [ ] Tulis isi (3 bab)
   - [ ] Tulis kesimpulan

3. **Revisi & Finalisasi** (3 hari)
   - [ ] Review struktur
   - [ ] Perbaiki typo
   - [ ] Format sesuai template
   - [ ] Submit

:::note Durasi Otomatis
Durasi phases dihitung mundur dari deadline. Total 11 hari (3+5+3). Pastikan deadline minimal 11 hari ke depan.
:::
