# Target Mingguan

Cara mencapai target 5 sesi belajar per minggu.

## Target: 5 Sesi/Minggu

PAM merekomendasikan **minimal 5 sesi belajar per minggu** untuk konsistensi akademik.

### Kenapa 5 Sesi?

- Cukup untuk momentum tanpa burnout
- Rata-rata 1 sesi/hari (weekdays)
- Fleksibel: bisa 2-3 sesi di weekend jika weekdays sibuk

## Strategi Mencapai Target

### 1. Fixed Schedule

Tentukan jadwal tetap, contoh:
- **Senin-Jumat**: 19:00-20:00 (1 sesi 50m/hari)
- **Sabtu**: 2 sesi (pagi + sore)

### 2. Pomodoro Stack

Stack beberapa Pomodoro:
- 3x Pomodoro (25m) = 1 sesi deep work equivalen

### 3. Weekend Catch-Up

Jika weekdays cuma dapat 3 sesi, kejar 2 sesi di weekend.

## Tips Konsistensi

1. **Same Time, Same Place** - Rutinitas fixed
2. **Start Small** - 25m lebih baik daripada 0
3. **Log Immediately** - Jangan tunda log sesi
4. **Visual Progress** - Cek dashboard tiap hari
